# WPTE 2026 Repo Prep

This note tracks the repo-side material for a certificates-focused decreasing-diagrams submission without changing anything under `paper/`.

## What the repo already supports

- Self-contained external confluence certificates checked inside Lean.
- A canonical compressed representation enforced by the strengthened external checker.
- Seven hand-written positive external benchmarks plus negative external regressions.
- A generated external family with scalable raw and compressed artifacts.
- Reproducible artifact export, checking, benchmarking, and roundtrip validation through `ddcert`.

## Submission-facing artifacts

- `external/`: seven hand-written benchmarks covering fan, shortcut, zigzag, funnel, and multi-peak shapes.
- `negative/`: invalid external artifacts for parser and checker regressions.
- `external-family/`: generated family artifacts together with size, timing, and roundtrip CSVs.
- `external-family-negative/`: generated invalid family artifacts that exercise canonical-compression and coverage failures.

## Recommended regeneration path

Use the workshop-prep script when you want a fresh, submission-scale bundle:

```bash
scripts/prepare-wpte-2026.sh
```

The script performs a full `lake build`, refreshes the decreasing-diagrams artifact directories, checks the largest generated family instance in the selected range, confirms the negative family artifacts are rejected, and copies the main CSV outputs into `artifacts/decreasing-diagrams/wpte-2026/`.

The current writing-ready outline is in `artifacts/decreasing-diagrams/wpte-2026/SUBMISSION-OUTLINE.md`.
The current EPTCS-format draft source is in `artifacts/decreasing-diagrams/wpte-2026/wpte2026-draft.tex`.

## Current thesis to support

The strongest repo-backed story is:

- proof-producing checking for self-contained decreasing-diagram certificates;
- a deterministic canonical compression layer for exported certificates;
- negative regressions showing that malformed, incomplete, or non-canonical artifacts are rejected; and
- a generated family that scales the checked artifact story beyond a few hand-written examples.
